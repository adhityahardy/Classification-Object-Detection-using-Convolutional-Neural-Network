About Project

Web ini merupakan web yang dapat melakukan deteksi kecepatan kendaraan dan identifikasi jenis kendaraan.

Cara kerja dari kecepatan kendaraan ini adalah menggunakan konsep image processing dalam python dengan menggunakan opencv dan tensorflow. Sistem ini menggunakan objek tracking untuk mendapatkan kecepatan kendaraan. Saat objek tracking tersebut menyentuh garis merah maka sistem akan mengetahui dari setiap kendaraan yang lewat.

Cara kerja dari deteksi kendaraan ini adalah menggunakan metode CNN (Convolutional Neural Network) dengan dataset cifar10. Dataset tersebut ditraining menggunakan google colab lalu hasil model training tersebut dijalankan menggunakan visual studio code. Jenis Kendaraan yang dideteksi adalah mobil dan truk.
